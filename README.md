HealthZone - Hospital Management System

Hospital Management System built with Django.

Features
- Patient Management
- Doctor Management
- Appointment Booking
- User Login and Signup
- Dashboard
- AI Chatbot
- REST API

Tech Stack

- Django
- PostgreSQL
- Django REST Framework
- Google Gemini AI

How to Install

1. pip install -r requirements.txt
2. python manage.py migrate
3. python manage.py runserver
